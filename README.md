# 🧮 Simple Calculator (Python CLI)

A simple command-line calculator built with Python.  
This program performs basic arithmetic operations:

- Addition
- Subtraction
- Multiplication
- Division (with zero-division protection)

---

## 🚀 Features

✅ Interactive terminal interface  
✅ Handles division by zero using loop  
✅ Beginner-friendly Python project  
✅ Clean and structured functions  

---

## 🛠️ Technologies Used

- Python 3

## ▶️ How to Run step by step
1. Open terminal on your kali linux
2. sudo apt install python3
3. git clone https://github.com/alamincyber/calculator
4. cd calculator
5. python calculator.py
